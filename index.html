2. 문서형(Document Type Definition) 선언

                <!DOCTYPE html>

3. 언어 / 문자셋(charset) 지정

              <html lang="ko">
            
              <meta charset="UTF-8">

4. http-equiv

웹 브라우저가 서버에 명령을 내리는 속성으로 name 속성을 대신하여 사용될 수 있으며, HTML 문서가 응답 헤더와 함께 웹 서버로부터 웹 브라우저에 전송되었을 때에만 의미를 갖는다. http-equiv 속성이 명시되었으면 content 속성 값 또한 필수이다.

              <meta http-equiv="X-UA-Compatible" content="IE=edge" />
            
              <meta http-equiv="refresh" content="3; url=이동하고자하는 웹페이지 주소">
            
              <meta http-equiv="Content-Script-Type" content="text/javascript">
            
              <meta http-equiv="Content-Style-Type" content="text/css">

5. 뷰포트

              <meta name="viewport" content="width=device-width, initial-scale=1.0">  # PC
            
              <meta name="viewport" content="width=device-width,initial-scale=1.0,minimum-scale=1.0,maximum-scale=1.0,user-scalable=no">  # MOBILE

6. SEO(검색엔진최적화)

              <meta name="title" content="홈페이지 제목">
            
              <meta name="keywords" content="대표키워드">
            
              <meta name="description" content="홈페이지 설명 페이지 정보요약">
            
              <meta name="author" content="페이지 저자">

7. OPEN GRAPH / TWITTER

              <meta property="og:type" content="website">
<meta property="og:site_name" content="사이트이름">
<meta property="og:title" content="title">
<meta property="og:description" content="where people build software.">
<meta property="og:image" content="이미지경로">
<meta property="og:url" content="도메인주소">
            
              <meta property="twitter:card" content="summary">
<meta property="twitter:site" content="사이트이름">
<meta property="twitter:title" content=타이틀">
<meta property="twitter:description" content="사이트설명">
<meta property="twitter:image" content="이미지경로">
<meta property="twitter:url" content="도메인주소">

8. Favicon

              <link rel="shortcut icon" type="image/x-icon" href="favicon.ico" > 
#IE root에 파일이 있을경우 생략가능
<link rel="icon" href="favicon.png">
<link rel="apple-touch-icon" href="favicon.png">

9. 숫자로 된 항목을 전화번호로 인식하는걸 방지

              <meta name="format-detection" content="telephone=no">
