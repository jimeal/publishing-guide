1. HTML

웹표준 검사하기

바로가기

들여쓰기

1탭 = 공백 4칸

주석

  • 주석기호와 주석내용 사이에는 공백한칸이 있어야 한다.
  • 시작과 종료 주석의 내용은 동일해야한다.
                <!-- 시작주석 -->
<!-- //종료주석 -->

링크 anchor

                <a href="#"></a>
<a href="ftp://"></a>
<a href="mailto://"></a>
<a href="파일경로" download ></a>
<a href="url" target="_blank" title="새창"></a>
anchor tag attributes
속성 의미 값
href 링크 폴더(절대경로/상대경로), 같은 문서내의 element ID, JS함수호출, ftp:, mailto:
target 링크 URL의 표시(브라우저 탭) 위치 _self(기본값), _blank
rel 현재 문서와 링크 URL의 관계Link Types
download 이 요소가 리소스를 다운로드하는 용도로 사용됨을 의미 boolean
  • 빈 링크의 href값은 #으로 통일한다
  • 이전버전 (XHTML1.0)에서는 블록 요소를 포함할 수 없었지만, HTML5에서는 포함할수 있게 되었다
  • <input>, <button>, 다른 link(a태그) 요소들은 포함할 수 없다
  • href의 링크가 새창(_blank)으로 열릴경우 툴팁보조(title)속성을 이용하여 새창으로 표시된다는걸 알려주어야한다

마크업 유의사항

  • p, table로는 레이아웃을 잡지 않는다.
  • p 태그 안에 p, div등 블럭요소를 넣을수 없다.
  • p 태그는 최하위 레벨에만 사용한다.
  • button 태그에는 type을 반드시 선언한다. (브라우저 별 기본값이 다름)
  • form태그 안에는 form태그가 들어갈수 없다.
  • label 태그안에 label 태그가 들어갈 수 없다.
  • id값은 중복해서 사용할수 없다.

기능에 맞는 태그사용 (anchor, button, input)

  • 외부링크는 a태그 사용
  • 페이지 내 특정내용을 지시하는 경우, a 태그와 해시(#)링크 값 사용
  • 페이지 내 동작하는 특수 기능은 button 태그 사용
  • 폼값 전송을 위한 기능은 submit타입의 input 또는 button 태그 사용

속성선언순서

속성선언순서
구분 속성
기능 href, src, type
이름 name, id
정보 value, title, alt
옵션 target, tabindex

역할 role
속성 aria-controls, aria-describedby, aria-labelledby, aria-popup, aria-modal 등
상태 checked, disabled, selected, required, aria-selected, aria-current, aria-hidden 등

2. CSS

웹표준 검사하기

바로가기

들여쓰기

1탭 = 공백 4칸

주석

  • 주석기호와 주석내용 사이에는 공백한칸이 있어야 한다.
                /* 주석 */

선택자 및 우선순위

  • 태그는 언제든지 다른 태그로 변경될 수 있으므로 태그 선택자로 스타일을 작성하지 않는다.
  • !important는 불가피한 상황을 제외하고는 사용하지 않는다.

공백, 줄바꿈

  • 선택자 간, 중괄호간 한칸의 공백을 사용한다.
  • 속성 선언시 콜론 (:) 뒤에 한칸의 공백을 사용한다.
  • CSS 코드 작성시 중괄호 ({, }) 사이를 줄바꿈한다.
  • 각각의 속성 작성시 줄바꿈 처리한다.
                .class {
    background-color: blue;
    background-repeat: no-repeat;             
}

숨김텍스트

숨김 텍스트 처리할 시에 text-indent: -9999px, top: -9999px, font-size: 0, display: none 등으로 처리하지 않는다.
해당 스타일은 스크린 리더기에서 읽히지 않거나, 초점 이슈가 존재할 수 있다

                .sr-only {
    overflow: hidden;
    position: absolute !important;
    clip: rect(0, 0, 0, 0);
    clip-path: inset(50%);
    width: 1px;
    height: 1px;
    margin: -1px;
}
